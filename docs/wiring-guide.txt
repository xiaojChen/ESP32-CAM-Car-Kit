# 接线指南

## 电源接线图

`
     2x18650 电池盒（串联 7.4V）
            │
       ┌────┴────┐
       │         │
     L298N     LM2596S
    12V输入    IN+  IN-
       │         │
       │     OUT+ ──→ ESP32-CAM V5 (调到5.0V)
       │     OUT- ──→ ESP32-CAM GND
       │         │
     GND ────────┴── ESP32-CAM GND (共地)
`

## 控制信号表

| ESP32-CAM GPIO | 连接 L298N | 功能 |
|----------------|------------|------|
| IO12 | IN1 | 电机 A 方向 1 |
| IO13 | IN2 | 电机 A 方向 2 |
| IO15 | IN3 | 电机 B 方向 1 |
| IO14 | IN4 | 电机 B 方向 2 |
| IO2 | ENA | 电机 A 速度 PWM（取下跳线帽） |
| IO33 | ENB | 电机 B 速度 PWM（取下跳线帽） |
| IO4 | 闪光灯 LED | 补光灯控制 |

## 电机接线

`
L298N OUT1 ─┬─ 左前电机 +
            └─ 左后电机 +
L298N OUT2 ─┬─ 左前电机 -
            └─ 左后电机 -

L298N OUT3 ─┬─ 右前电机 +
            └─ 右后电机 +
L298N OUT4 ─┬─ 右前电机 -
            └─ 右后电机 -
`

## 重要提示

1. 共地：电池 GND = L298N GND = LM2596S OUT- = ESP32-CAM GND
2. ENA/ENB 跳线帽必须取下，否则不能调速
3. LM2596S 先调好 5.0V 再接 ESP32（万用表测量）
4. 不要同时接 USB 和 LM2596S 供电
5. 电机线正反接反会导致方向反，交换两根线即可
